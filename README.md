# Social Media Downloader

X (Twitter), YouTube ve Instagram'dan resim, video ve medya içeriklerini kolayca indirin.

![Version](https://img.shields.io/badge/version-1.1.0-blue)
![Manifest](https://img.shields.io/badge/Manifest-V3-green)
![License](https://img.shields.io/badge/license-MIT-orange)

## Ozellikler

### X (Twitter)
- Tweet resimlerini orijinal kalitede indir
- Tweet videolarini tum kalite secenekleriyle indir (1080p, 720p, 480p, 360p)
- GIF'leri indir
- Profil banner resimlerini indir
- Otomatik video URL yakalama (API interceptor)

### YouTube
- Video thumbnail'lerini 4 farkli kalitede indir
- Video linkini kopyala

### Instagram
- Gonderi resimlerini en yuksek kalitede indir
- Video ve Reel icerikleri indir
- Story resimlerini indir
- Carousel (coklu resim) destegi

## Kurulum

### Chrome / Edge / Brave / Opera

1. Bu repoyu ZIP olarak indirin:
   - Yesil **"Code"** butonuna tiklayin → **"Download ZIP"** secin
   - Veya [Releases](../../releases) sayfasindan son surumu indirin

2. ZIP dosyasini bir klasore cikartin

3. Tarayicinizda eklenti sayfasina gidin:
   - **Chrome**: `chrome://extensions`
   - **Edge**: `edge://extensions`
   - **Brave**: `brave://extensions`
   - **Opera**: `opera://extensions`

4. Sag ustten **"Gelistirici modu"** nu acin (Developer mode)

5. **"Paketlenmemis oge yukle"** (Load unpacked) butonuna tiklayin

6. ZIP'ten cikardiginiz klasoru secin

7. Eklenti tarayici cubugunda gorunecek!

### Firefox

> Firefox icin Manifest V2 uyumlu ayri bir build gerekmektedir. Yakinda eklenecek.

## Kullanim

### Otomatik Indirme Butonlari
Eklenti desteklenen sitelerde medya iceriklerinin uzerine otomatik olarak indirme butonu ekler. Butona tiklayarak icerik indirebilirsiniz.

### Popup ile Indirme
1. Desteklenen bir siteye gidin (X, YouTube, Instagram)
2. Tarayici cubugundaki eklenti ikonuna tiklayin
3. Bulunan medya listesini gorun
4. Tek tek veya toplu indirin
